Module 1: Launching into Cyber Security

A 12-Unit Module that Started on March 2022 and was Covered in 12 Weeks by Stelios Sotiriadis

Summary

This Module provided a high-level Introduction/Overview of the topics covered in the other Modules of the Master's programme.

It also covered the concepts and principles in the field of Cyber Security, the governance and ethical frameworks and competencies required as a Cyber Security professional.

Skills Acquired during this Module

  1. Awareness of the Ethical and Professional Responsibilities as a Cyber Security Professional.

  2. Gained Knowledge of the Cyber Security-based Challenges and its Solutions: MFA, Firewalls, etc.

  3. Using Python programming so as to create Cyber Security code solutions: Input and Password validator, Code and Password generator, etc.

  4. Gained Knowledge of the effects of Cyber Attacks in an organisation or society: loss of trust, financial loss, data loss, etc.

  5. Gained Knowledge of the Cyber Attacks and Techniques: Malwares, Denial of Service, Keylogging, etc.

  6. Gained the ability to Design appropriate Cyber Security Solutions: by using threat models like Attack trees, OWASP, STRIDE,DREAD, etc

  7. It provided me with Aspects that I would likely encounter in Cyber Security professionalism.

  8. I was able to engage, experience and envision the current as well as the future development of Cyber Security discipline.

  9. Gained knowledge on secure web development and also web development using HTML,CSS.
